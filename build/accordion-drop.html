Ответы на часто задаваемые вопросы:
  • ГИС ЖКХ
    • Чтение данных с ГИС ЖКХ
      Возможность импорта данных, размещенных на сайте ГИС ЖКХ, в АСУ Жилищный Стандарт (по подписке, активированной для АСУ Жилищный Стандарт)
    • Запись данных на ГИС ЖКХ через API

      Возможность экспорта данных на сайт ГИС ЖКХ, по подписке, активированной для ИС Жилищный Стандарт. пп. 1-2;3.1-3.3; 7-8; 14-15;17-20 Раздела 10 Приказа 74-114

      Возможность экспорта данных на сайт ГИС ЖКХ, по подписке, активированной для ИС Жилищный Стандарт. пп. 3.2; 3.4-3.6; 4.1; 5.1-5.4; 9-10;12 *

    • Автоматическое заполнение шаблонов импорта по формату ГИС ЖКХ
      Возможность выгрузки данных, размещенных в АСУ Жилищный Стандарт, в шаблоны формата xlsx, соответствующие актуальным шаблонам ГИС ЖКХ для полуавтоматического импорта данных на ГИС ЖКХ по Управляющей Организации
    • Импорт данных в АСУ с результирующих файлов-шаблонов ГИС ЖКХ
      Возможность импортировать в АСУ Жилищный Стандарт данные с файлов, генерируемых ГИС ЖКХ в процессе импорта данных, и сохранения в АСУ Жилищный Стандарт уникальных кодов объектов для последующего оперирования ими
    • Печатная форма анкеты Дома
