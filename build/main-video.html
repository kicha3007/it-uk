Какой функционал есть у ПО?
Возможности АСУ «Жилищный Стандарт» по автоматизации:
  • автоматическое наполнение сайта Реформа ЖКХ по API
  • автоматическое наполнение сайта ГИС ЖКХ по API
  • возможность выгрузки данных в шаблоны xlsx для последующего импорта на ГИС ЖКХ
  • максимальное удобство при заполнении данных: автоматическая генерация данных; массовая работа с данными с помощью шаблонов xlsx
  • автоматическое представление данных по 882-П на сайте УК
